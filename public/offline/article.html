April 27, 2009

 

If you want to see this in action you should be loading this through your iPhone. The first time you load this the files specified in the manifest file will cache to your phone.

Enable Airplane mode to cut all network connections and reload this page. Watch as it loads without a problem.

If you didn't visit this page before you disabled your network connections this page would still load as it was declared in the manifest file.

The page that the manifest file originates from does not need to be declared in the manifest file. It will automatically be included in the cache.
